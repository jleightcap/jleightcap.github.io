Embedded YouTube videos

For a YouTube video URL https://youtu.be/rB83DpBJQsE, use the video ID (rB83DpBJQsE) and youtube.com/embed to make an embedded YouTube player. If just using an iframe, the video looks fine on desktop, but exceeds the text for small widths. To fix this, the size of the embedded video changes. This uses CSS, so in config.toml add customCSS = ["css/videos.css"], where the file videos.css contains 

.video-container {
  position: relative;
  padding-bottom: 56.25%;
  padding-top: 30px; height: 0; overflow: hidden;
}

.video-container iframe,
.video-container object,
.video-container embed {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
This adds the CSS necessary, so 
<div class="video-container">
<iframe width="853" height="480" src="https://www.youtube.com/embed/rB83DpBJQsE"
  allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"
  frameborder="0"
  allowfullscreen>
</iframe>
</div>
