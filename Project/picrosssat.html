Translating A Puzzle

Writing expressions in Disjunctive Normal, or Sum of Products (SOP), form is simple:

x = {4}:
(a ∧ b ∧ c ∧ d)
x = {3}:
(a ∧ b ∧ c ∧ ¬d)∨(¬a ∧ b ∧ c ∧ d)
x = {1, 2}:
(a ∧ ¬b ∧ c ∧ d)
x = {2, 1}
(a ∧ b ∧ ¬c ∧ d)
x = {2}:
(a ∧ b ∧ ¬c ∧ ¬d)∨(¬a ∧ b ∧ c ∧ ¬d)∨(¬a ∧ ¬b ∧ c ∧ c)
x = {1, 1}
(a ∧ ¬b ∧ c ∧ ¬d)∨(a ∧ ¬b ∧ ¬c ∧ d)∨(¬a ∧ b ∧ ¬c ∧ d)
x = {1}:
(a)∨(b)∨(c)∨(d)

However, SAT-CNF solvers require Conjunctive Normal, or Product of Sums (POS), form. This translation in the worst case is O(2n):

x = {4}:
(a)∧(b)∧(c)∧(d)
x = {3}:
(¬a ∨ ¬d)∧(a ∧ d)∧(b)∧(c)
x = {1, 2}:
(a)∧(¬b)∧(c)∧(d)
x = {2, 1}
(a)∧(b)∧(¬c)∧(d)
x = {2}:
(¬a ∨ b)∧(¬a ∨ ¬c)∧(a ∨ c)∧(¬b ∨ ¬d)∧(b ∨ d)∧(c ∨ ¬d)
x = {1, 1}
(¬a ∨ ¬b)∧(a ∨ b)∧(¬b ∨ ¬c)∧(¬c ∨ ¬d)∧(c ∨ d)
x = {1}:
(a ∨ b ∨ c ∨ d)
