Code Blocks

Add pygmentsUseClasses = true to config.toml. This is default in hermit. To display a code block, the plaintext in *.md renders as

def fibonacci(n):
   if n <= 1:
       return n
   else:
       return(fibonacci(n-1) + fibonacci(n-2))

TeX

Can't use .md, use .mmark as extension of document. Add to header of *.mmark markup = mmark. Add to bottom of footer.html (which is loaded on every page) 

<script src="//yihui.name/js/math-code.js"></script>
<script async
  src="//mathjax.rstudio.com/latest/MathJax.js?config=TeX-MML-AM_CHTML">
</script>
This uses MathJax, but it might be worthwhile to swap to KaTeX for faster rendering.

Math Mode

To display Math Mode TeX, the plaintext in *.mmark

`\[ \vec{\nabla} \cdot \vec{E} = \frac{\rho}{\varepsilon_0} \]`
`\[ \vec{\nabla} \cdot \vec{B} = 0 \]`
`\[ \vec{\nabla} \times \vec{E} = - \frac{\partial \vec{B}}{\partial t} \]`
`\[ \vec{\nabla} \times \vec{B} = \frac{1}{c} \frac{\partial \vec{E}}{\partial t} \]`

renders as \[ \vec{\nabla} \cdot \vec{E} = \frac{\rho}{\varepsilon_0} \] \[ \vec{\nabla} \cdot \vec{B} = 0 \] \[ \vec{\nabla} \times \vec{E} = - \frac{\partial \vec{B}}{\partial t} \] \[ \vec{\nabla} \times \vec{B} = \frac{1}{c} \frac{\partial \vec{E}}{\partial t} \]

In-Line

To display in-line TeX, the plaintext in *.mmark 

textleft `\( \det(\lambda I - \textbf{A}) = 0\)` textright
renders as
textleft \( \det(\lambda I - \textbf{A}) = 0\) textright

Embedded YouTube videos

Combining Images and Links

For a YouTube video URL https://youtu.be/rB83DpBJQsE, use the video ID (rB83DpBJQsE) and img.youtube.com to make a clickable image. 

[![Video Text](https://img.youtube.com/vi/rB83DpBJQsE/0.jpg)](https://youtu.be/rB83DpBJQsE)
renders as This has the clear disadvantage of not being embedded within the page, however works more universally.

Using HTML directly

For a YouTube video URL https://youtu.be/rB83DpBJQsE, use the video ID (rB83DpBJQsE) and youtube.com/embed to make an embedded YouTube player.

<div style="text-align:center">
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/rB83DpBJQsE"
  frameborder="0"
  allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
</div>

This looks much nicer on desktop and is more functional, but the width exceedes the text on mobile.
